
Sou desenvolvedora front-end apaixonada por criar **interfaces que ficam na memória** — onde código preciso encontra design com identidade. Acredito que a experiência do usuário começa nos detalhes: da escolha tipográfica ao timing de uma transição.

Minha abordagem combina **rigor técnico** com sensibilidade criativa. Transformo ideias complexas em produtos digitais elegantes que encantam usuários e geram resultados reais.
